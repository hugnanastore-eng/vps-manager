多站点WordPress

在单个VPS上创建无限量网站。自动安装WordPress，创建数据库，配置Nginx，SSL — 只需输入域名。

📋 每个选项做什么？

✅ 添加网站 — 输入域名 → 自动创建文件夹、数据库、安装WordPress、配置Nginx。不影响其他站点。

⚠️ 删除网站 — 先创建完整备份（文件+数据库），然后才删除Nginx配置。备份保存在 /backup/removed/。数据不会丢失。

✅ 添加重定向 — 将旧域名重定向到新域名。不修改数据。

✅ 修复权限 — 重置文件权限（755目录，644文件）。不删除任何东西。

✅ 维护模式 — 一键开启/关闭维护页面。

✅ 插件管理 — 查看、启用、禁用插件。不删除插件。

智能磁盘清理 新

先预览再清理。删除旧日志、journal、包缓存、WP修订版本、临时数据、垃圾评论。精确显示回收空间。

📋 详情：不会删除网站或文章！

🛡️ 此功能仅清理垃圾文件。不会删除WordPress网站或文章。

✅ [1] 旧日志 — 删除/var/log中的轮转日志(*.gz, *.1, *.2...)。这些是旧副本，不是活动日志。

✅ [2] 包缓存 — 清除dnf/apt下载缓存。类似清除浏览器缓存——安全。

✅ [3] WP临时数据 — 删除WordPress数据库中的临时缓存。WordPress会自动重建。

✅ [4] WP修订版 — 删除文章的旧草稿版本。已发布的文章不受影响。

⚠️ [5] 旧备份 — 删除30天前的备份前会询问确认。

✅ [6] 临时文件 — 显示/tmp大小，不自动删除。

💡 始终先运行预览模式，再询问是否执行。

WAF与安全

Web应用防火墙（WAF）拦截SQL注入、XSS、路径遍历。结合fail2ban + 安全头 + 文件完整性监控。

📋 What each option does

✅ WAF status — Shows how many malicious requests were blocked today. View only.

✅ Integrity check — Compares WordPress core files against official checksums. Reports suspicious changes but does NOT auto-delete.

✅ fail2ban status — Shows which IPs are currently banned. View only.

✅ Ban/Unban IP — Manually block or unblock a specific IP address.

✅ Recent attacks — Shows last 20 blocked requests. View only.

常见问题解答

脚本安全吗？

安全。全部源码开放，运行前可检查。脚本不会向外发送数据，除了Telegram告警（如果您配置了的话）。

支持哪些操作系统？

Rocky Linux 8/9（推荐）、AlmaLinux 8/9、Ubuntu 22.04、Ubuntu 24.04。VPS至少需要2GB内存和10GB磁盘空间。

与HocVPS脚本有什么不同？

VPS一键v2.1优于HocVPS，拥有9大模块：分表数据库备份、WordPress自动更新带回滚、SSH密钥管理器、域名健康仪表盘、暂存站点、访问分析、资源告警Telegram。HocVPS脚本不具备这些功能。

需要Linux专业知识吗？

不需要深入专业知识。只需会SSH登录VPS即可。管理菜单完全交互式，标注清晰 — 选择数字即可操作。

能从HocVPS迁移吗？

可以。只需备份旧VPS数据（数据库+wp-content），在新VPS上安装脚本，然后导入。脚本在数据库菜单中内置SQL导入功能。
