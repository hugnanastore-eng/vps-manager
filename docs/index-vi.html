Multi-site WordPress

Tạo không giới hạn website trên 1 VPS. Tự động cài WordPress, tạo database, cấu hình Nginx, SSL — chỉ cần nhập tên domain.

📋 Chi tiết: từng option làm gì?

✅ Thêm website mới — Nhập domain → tự tạo thư mục, database, cài WordPress, cấu hình Nginx. Không ảnh hưởng site khác.

⚠️ Xóa website — TẠO BACKUP TRƯỚC (file + database), sau đó mới xóa config Nginx. Backup lưu tại /backup/removed/. Dữ liệu KHÔNG bị mất.

✅ Thêm redirect — Chuyển hướng domain cũ sang domain mới. Không sửa dữ liệu.

✅ Sửa quyền file — Reset ownership và chmod (755 thư mục, 644 file). KHÔNG xóa gì.

✅ Chế độ bảo trì — Bật/tắt trang bảo trì. Nhấn 1 lần bật, nhấn lại tắt.

✅ Quản lý plugin — Xem, bật, tắt plugin bằng đổi tên thư mục. KHÔNG xóa plugin.

Resource Alert + Telegram MỚI

Giám sát RAM, Disk, CPU, Swap theo ngưỡng tùy chỉnh. Cảnh báo Telegram tự động khi vượt mức, cooldown 30 phút tránh spam. Setup cron 1 click.

📋 Chi tiết: từng option làm gì?

✅ Xem trạng thái hiện tại — Hiển thị RAM %, Disk %, CPU, Swap, top process. Màu sắc: 🟢 bình thường, 🟡 cảnh báo (>70%), 🔴 nghiêm trọng (>90%).

✅ Cảnh báo Telegram (cron) — Chạy mỗi 5 phút. Chỉ gửi khi vượt ngưỡng. Cooldown 30 phút tránh spam. Chỉ theo dõi — KHÔNG thay đổi gì trên server.

Disk Cleanup Thông Minh MỚI

Dry-run trước để xem sẽ dọn gì. Xóa logs cũ, journal, package cache, WP revisions, transients, spam comments. Hiển thị chính xác dung lượng thu hồi được.

📋 Chi tiết: KHÔNG xóa website hay bài viết!

🛡️ Tính năng này chỉ dọn file rác. KHÔNG xóa WordPress, bài viết hay dữ liệu của bạn.

✅ [1] Log cũ — Xóa file log đã xoay vòng (*.gz, *.1, *.2...) trong /var/log. Đây là bản cũ, không phải log đang dùng.

✅ [2] Cache gói cài đặt — Xóa cache dnf/apt đã tải. Giống xóa cache trình duyệt — an toàn.

✅ [3] WP transients — Xóa cache tạm trong database WordPress. WordPress tự tạo lại ngay.

✅ [4] WP revisions — Xóa bản nháp cũ của bài viết. Bài đã ĐĂNG vẫn nguyên vẹn.

⚠️ [5] Backup cũ — Hỏi trước khi xóa backup quá 30 ngày. Bạn phải xác nhận.

✅ [6] File tạm — Hiển thị kích thước /tmp. Không tự xóa.

💡 Luôn chạy dry-run trước (xem trước) rồi mới hỏi có muốn thực hiện không.

Firewall Hardening

Kernel tuning SYN flood, UFW/firewalld auto-config, fail2ban 5 jails, SSH hardening, Nginx DDoS protection, auto-update OS, change root password.

📋 Chi tiết: từng option làm gì?

✅ Kernel tuning — Tối ưu tham số chống SYN flood. Không ảnh hưởng dịch vụ.

✅ UFW/firewalld — Tự cấu hình firewall cho phép SSH(22), HTTP(80), HTTPS(443). Chặn port khác.

✅ fail2ban — Thiết lập 5 jail (SSH, Nginx) tự chặn IP brute-force. Chỉ chặn tạm thời.

✅ SSH hardening — Tắt đăng nhập password (chỉ khi đã có SSH key), thay đổi cài đặt mặc định.

✅ Chống DDoS Nginx — Giới hạn tốc độ request. Người dùng thường không bị ảnh hưởng.

🛡️ Backup toàn bộ config trước khi thay đổi. Test SSH trước khi hoàn tất.

Multi-IP Management

VPS có nhiều IP? Gán IP riêng cho từng website qua nginx listen directive. Hiện tất cả IPs, bind/unbind, verify. Auto-backup + rollback.

📋 Chi tiết: từng option làm gì?

✅ Xem tất cả IP — Liệt kê tất cả IP trên VPS. Chỉ xem.

✅ Gán IP cho site — Gán IP cụ thể cho website qua Nginx listen directive. Tự backup config trước khi thay đổi.

✅ Bỏ gán IP — Trở về mặc định (listen trên tất cả IP). Tự backup + rollback nếu lỗi.

✅ Kiểm tra — Test xem binding hoạt động đúng không.

FAQ

Câu Hỏi Thường Gặp

Script có an toàn không?

Có. Toàn bộ mã nguồn mở, có thể kiểm tra trước khi chạy. Script không gửi dữ liệu ra ngoài, ngoại trừ Telegram alert (nếu bạn cấu hình).

Hỗ trợ hệ điều hành nào?

Rocky Linux 8/9 (khuyến nghị), AlmaLinux 8/9, Ubuntu 22.04, Ubuntu 24.04. VPS cần ít nhất 2GB RAM và 10GB ổ đĩa.

Khác gì HocVPS Script?

Mạnh hơn HocVPS ở: WAF bảo mật, file integrity monitor, Telegram cảnh báo real-time, auto-update plugin có rollback, VPS cluster sync, SEO tự động, speed test tích hợp. Giữ nguyên tính dễ dùng.

Có cần biết Linux để dùng không?
